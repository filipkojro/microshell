# microshell

## Run / build

### Building (linux / mac)
u need `gcc`, `make`
```sh
git clone https://github.com/filipkojro/microshell.git
cd microshell
make
```

### running (linux / mac)
While running you have to be in microshell directory.
```sh
./microshell
```
