## warning todo

### running (linux / mac)
u need `gcc` and `make`

while running the first time you probably should use commands below so config will generate correctly
```sh
cd /your/designated/location/for/microshell
git clone https://github.com/filipkojro/microshell.git
cd microshell
make
./microshell
```
